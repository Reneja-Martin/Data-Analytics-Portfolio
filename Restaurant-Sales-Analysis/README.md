# Marketplace Performance Analysis  
## Uber Eats vs SkipTheDishes

### Project Overview
This project analyzes restaurant sales performance across Uber Eats and SkipTheDishes to compare order volume, sales trends, and platform performance.

### Business Problem
Restaurants operating across multiple delivery platforms need visibility into marketplace performance to better understand customer ordering trends and sales activity.
