# Complaint Operations Dashboard

## Project Overview
This Power BI dashboard analyzes operational complaint trends across transportation and warehouse functions to identify complaint patterns, operational challenges, and recurring root causes. The project was designed to improve visibility into complaint activity and support operational reporting through interactive data visualization.

---

## Business Questions Explored
- Which locations generate the highest complaint volume?
- Are complaints driven more by transportation or warehouse operations?
- Which carriers contribute the most transportation complaints?
- What are the primary root causes behind complaint activity?
- Which operational trends require the most attention?
