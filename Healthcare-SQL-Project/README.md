# Healthcare SQL Project

## Project Overview
This project analyzes healthcare appointment and patient data using SQL to identify operational trends and improve reporting visibility.

## Business Problem
Healthcare organizations need data insights to better understand appointment attendance, scheduling efficiency, and patient trends.

## Tools Used
- SQL
- Excel
- Power BI

## Skills Demonstrated
- SQL Queries
- Data Cleaning
- Data Analysis
- Aggregations
- Joins
- Reporting

## Example Business Questions
- Which departments had the highest no-show rates?
- What times of day had the most missed appointments?
- Did waiting time impact attendance rates?
- Which appointment types had the highest completion rates?
